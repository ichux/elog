STEP 1
# To determine IP address of host
1. make up
2. docker run -it --rm edock netstat -nr | grep '^0\.0\.0\.0' | awk '{print $2}'
3. copy out the address from your terminal and use it

STEP 2
# Run bash to determine IP address of host
1. make bash
2. netstat -nr | grep '^0\.0\.0\.0' | awk '{print $2}'
3. copy out the address from your terminal and use it

EXTRA
# Use curl from entry
1. add the line below to Dockerfile
2. ENTRYPOINT ["/usr/bin/curl"]
3. docker run --rm edock https://www.google.com

CLEAN UP
### To clean up
make down
